Amateur Swimming Association of Jamaica

 

BLP8.4 Membership Renewal Procedure using HyTek Team Manager

 

 

Teams or individuals using HyTek TM5.0 can submit their list of swimmers for ASAJ registration by creating an athlete roster file and athlete roster report (pdf format) and emailing BOTH to the ASAJ Office. The athlete roster report will be used as the official registration document to validate the correct names have been imported into the ASAJ database. This method can ONLY be used if the team/individual has entered at least the following information in their database.

 

Swimmer’s Data (Surname, First name, Middle name, Preferred name (if applicable), Sex, Date of birth, Citizenship/Country, Other Federation (if any), Club, home phone, cell phone, email)

At least one parent’s data as Primary Contact (Mother/Father/Guardian Name, cell phone, work phone, email)

 

 

 

 

Procedure to create the Hytek export file (Athlete Roster)

To avoid sending the complete list of athletes in each submission, we suggest that new registrations be uniquely identified by creating and using a Filter Group eg School Year=FR. You must remove this filter from the athlete data as soon as his/her registration has been done.

 

 

Procedure to create the Hytek report file (Athlete Roster)

 

 

Select the relevant Team and Group Filters. Click [Create Report]. Print the report to a pdf printer. Hard copy can also be created by sending the report to a regular printer.
