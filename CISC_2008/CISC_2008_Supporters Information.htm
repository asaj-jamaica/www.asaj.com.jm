Two Supporters’ Packages are being offered:

OPTION 1:       Advance Package        PRICE: USD130.00 per adult

                                                                              USD100.00 per child

 

OPTION 2:        Local Package         PRICE: USD150.00 per adult

                                                               USD120.00 per child

Both packages include:

  • Two daily round trips to and from the pool from designated departure points
  • Entry Pass for all events and Closing Ceremony (Season ticket/Wrist bracelet)
  • Commemorative Magazine
  • Daily Heat/Score Sheets

In addition, the Advance Package include Round trip transportation - Airport/Hotel/Airport. Registration forms for this package must be returned to the ASAJ offices by June 14th 2008 with full payment. This has been extended to June 30, 2008.

ID Pass (Wrist bracelet) must be worn at all times. This bracelet will be used as your pass to board the buses, admission to the pool which will include heat/finals/score sheets.

Airport to Hotel transfers will be offered, if reservation is made before arrival.

One Way Transfer:    USD20.00 per adult / USD10.00 per child

 

Supporters are been offered shuttle service among all competition venue, departing/returning to the main venue at the National Stadium pool at a cost of USD60.00 per adult / USD20.00 per child for the duration of the competition. The service will not stop at any hotel or the games village.

All monies must be paid to the ASAJ - See Supporter package registration form for payment details.
